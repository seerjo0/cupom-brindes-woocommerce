Enhance WooCommerce coupons with automatic free gifts and personalized messages to boost conversions and customer satisfaction.

== Description ==

= Powerful Coupon Features =
Transform your WooCommerce coupons into powerful marketing tools:

* **Free Gift Automator** - Add products to cart automatically when coupons are applied
* **Message Customizer** - Replace default messages with branded content
* **Smart Coupon Engine** - Works seamlessly with all WooCommerce features

= Perfect For =
- "Free gift with purchase" campaigns
- Loyalty reward programs
- Seasonal promotions
- Abandoned cart recovery

= Why Choose This Plugin? =
- Lightweight (no bloat)
- 100% WooCommerce native integration
- No coding required
- Mobile-friendly
- Translation ready

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/woocommerce-free-gifts` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to WooCommerce > Coupons to configure your free gifts and messages

== Frequently Asked Questions ==

= Can I offer multiple free gifts with one coupon? =
Currently each coupon can offer one free product. For multiple gifts, create separate coupons.

= Will this work with variable products? =
Yes! You can select any product type as your free gift.

= Can customers remove the free product? =
Yes, they can remove it like any other cart item, but the coupon will remain applied.

= Does this work with all WooCommerce themes? =
The plugin uses standard WooCommerce hooks and should work with all compliant themes.
